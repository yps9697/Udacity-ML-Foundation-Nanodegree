Investigating a Dataset : TMDb movie data

Introduction

The TMDb movie dataset contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue. Also the Attributes Data set includes id , imdb_id , popularity , budget , revenue , original_title, cast , homepage, director tagline , keywords , overview , runtime , genres , production_companies , release_date , vote_count , vote_average release_year , budget_adj , revenue_adj.

2.How does budgets,revenues and runtimes changes throughout different year?

Ques1: Relation Between vote_average and revenue? Answer:Due to little positive association to each other as Pearson correlation coefficient is equal to 0.2271.

Note: As Pearson correlation coefficient can take a range of values from +1 to -1. 0 indicates that there is no relation between the two given variables. A value greater than 0 means a positive association. that is, as the value of one variable increases, so does the value of the other variable. A value less than 0 indicates a negative association.

#vote_avg variable contain vote_average column dataset
vote_avg = read['vote_average'] 

#rev variable contain revenue column dataset
reve = read['revenue']

#pearson correlation coefficient
vote_std = (vote_avg - vote_avg.mean())/vote_avg.std(ddof = 0)
reve_std = (reve - reve.mean())/reve.std(ddof = 0)

pearson_c_cof = (vote_std * reve_std).mean()

print("Pearson correlation coefficient:" + str(pearson_c_cof))
